# Imagens das Configurações NVIDIA

Esta pasta contém os prints das configurações do Painel de Controle NVIDIA.

## Configurações Gerais (FPS + Latência)
Adicione os seguintes prints nesta pasta:
- `config_geral_1.png` - Primeira tela de configurações 3D
- `config_geral_2.png` - Segunda tela de configurações 3D
- `config_geral_3.png` - Terceira tela de configurações 3D

## Configurações de Cores
Adicione os seguintes prints nesta pasta:
- `config_cores_1.png` - Tela de ajuste de cores
- `config_cores_2.png` - Configurações adicionais de cores

## Como adicionar as imagens:
1. Tire os prints das configurações no Painel de Controle NVIDIA
2. Renomeie os arquivos conforme especificado acima
3. Copie os arquivos para esta pasta (`data/nvidia/`)
4. O bot irá detectar e exibir automaticamente as imagens

## Formato recomendado:
- Formato: PNG
- Resolução: Mínimo 800x600
- Qualidade: Alta (para melhor visualização no Discord)
